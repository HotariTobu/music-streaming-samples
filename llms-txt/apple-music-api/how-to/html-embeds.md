# HTML Embeds

Generates Apple Music web player HTML to embed into your web page.

## Use Marketing Tools

Go to our [Apple Music Marketing Tools](https://tools.applemediaservices.com/apple-music) to generate and copy the embed code for embedded player. The embed code will give you a preview player in an iFrame with preset styles.

In the `Preview Player` section, you can adjust the height and width of the iFrame, and click the `Copy Embed Code` button to generate the embed code.

## Embed Examples

The following content types can be embedded:
- Song
- Album
- Playlist
- Music Video

Visit [Apple Music Marketing Tools](https://tools.applemediaservices.com/apple-music) to generate embed codes for your content.
