# Support

For help with issues you might run into while developing your app using MusicKit on the Web or the Apple Music API, you can use the following resources:

## Developer Forums

[Developer Forums](https://developer.apple.com/forums/tags/musickit-js)

Ask questions or help out other developers using MusicKit.

## Feedback Assistant

[Feedback Assistant](https://feedbackassistant.apple.com)

Report bugs or unexpected behavior.

## System Status

[System Status](https://developer.apple.com/system-status/)

Check if the services you are leveraging have any known issues.
